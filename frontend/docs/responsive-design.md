# Responsive Design

## Breakpoints
- **Mobile:** 0-767px
- **Tablet:** 768px-1023px
- **Desktop:** 1024px+

## Mobile Adaptations (≤767px)
- Single column opportunity grid
- Vertical navigation stack
- Reduced padding (2rem → 1rem)
- Smaller typography (h1: 2.5rem → 1.75rem)
- Stacked metrics (2-col → 1-col)
- Simulation form stacks vertically

## Tablet Adaptations (768-1023px)
- 2-column opportunity grid
- Moderate padding (1.5rem)
- Adjusted typography (h1: 2rem)
- 2-column metrics maintained
- Simulation forms stack for usability

## Desktop (≥1024px)
- Multi-column grids (auto-fill, minmax 350px)
- Full spacing and typography
- Maximum container width: 1200px
- Horizontal navigation
- Multi-column simulation layout

## CSS Approach
- Mobile-first media queries
- CSS Grid for layouts
- Flexbox for navigation and controls
- No hardcoded pixel widths in components
